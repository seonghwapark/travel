# ✈️ travel

국내선 항공편 통합 검색 웹앱. 여러 항공사 일정을 한 화면에서 비교할 수 있다.

## 주요 기능

- 출발지·도착지·날짜 입력으로 국내선 항공편 검색
- 여러 항공사 운임 및 시간 비교
- 직관적인 웹 UI

## 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | React (Vite) |
| Backend | Python (FastAPI) |

## 실행 방법

```bash
# 백엔드 실행
start-backend.bat

# 프론트엔드 실행
start-frontend.bat

# 또는 한번에
start.bat
```

## 프로젝트 구조

```
travel/
├── frontend/       # React 앱
├── backend/        # FastAPI 서버
└── start.bat       # 통합 실행 스크립트
```
